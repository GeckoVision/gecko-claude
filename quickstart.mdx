A **Receipt** is Gecko's answer on a proposed move: a verdict (act / pass /
defer), the opinions that disagreed, and the sources. Here's how to get one.

## Option A — add Gecko to your agent (one line)

Gecko runs a hosted server, so there's nothing to install. In Claude Code:

```bash
claude mcp add --transport http gecko https://api.geckovision.tech/mcp
```

Now your agent has Gecko's tools. Ask it to check a move before it acts:

```text
Before you do anything, ask Gecko whether depositing USDC into Kamino is sound
right now, and show me the verdict and the dissent.
```

You get back a verdict with the reasoning and the dissenting view attached —
the agent's own decision, now checked by an outside layer.

Any MCP-compatible agent works (Claude, OpenAI, LangChain, and others):

```json
{ "mcpServers": { "gecko": { "url": "https://api.geckovision.tech/mcp" } } }
```

## Option B — call the API directly

```bash
# 1. Mint a session for a wallet (we record the public address only)
curl -s -X POST https://api.geckovision.tech/v1/onboarding/link \
  -H 'Content-Type: application/json' \
  -d '{"wallet_address":"<your-wallet-address>"}'        # → { session_token }

# 2. Ask for a verdict
curl -s -X POST https://api.geckovision.tech/v1/research \
  -H "Authorization: Bearer <session_token>" \
  -H 'Content-Type: application/json' \
  -d '{"idea":"deposit USDC into Kamino","protocol":"kamino","vertical":"dex"}'
```

The full contract is at [the verdict API](/api/verdict), with a live "try it"
console in the **API Playground** tab.

<Note>During early user-testing, calls run **free in test mode** — the whole path works end to end and no money moves. See [Pricing](/pricing).</Note>

## Option C — run the proof agent yourself

Want to watch the verification layer drive a live (paper-trading) agent on your
own machine? Install the skills and launch the local control plane:

```bash
curl -fsSL https://app.geckovision.tech/install.sh | bash
bash launch_control_plane.sh        # serves on :8271, paper mode by default
curl localhost:8271/healthz
```

Then connect the app as a control surface:

```bash
export GECKO_AGENT_CONTROL_URL=http://localhost:8271
```
