| Tier | Price | What you get |
|---|---|---|
| **Self-hosted** | Free | Run the agent + control plane on your machine. You pay only for oracle calls. |
| **Hosted** | ~$0.04 / active agent-hour | We run the agent; you connect the app. |
| **Verdict** | ~$0.50 / verdict | A grounded oracle verdict (panel + dissent + citations + on-chain receipt). |

Billing is **pay-as-you-go over x402** — every paid call carries a Solana transaction signature. No subscription, no seat licenses.

## The hurdle principle
A self-funding-yield tier is on the roadmap: the profit vault's yield offsets hosting cost, so an agent that clears its **hurdle rate** effectively runs itself. See [the safety monitor](/vault/safety-monitor).

<Note>Stub mode (`X402_MODE=stub`) is free and exercises the full payment path during testing.</Note>
