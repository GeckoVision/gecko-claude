We publish what's proven and flag what isn't. Same discipline as the verdicts.

## Live now

- **The verdict** — grounded `act` / `pass` / `defer` with surviving dissent + citations.
- **Pre-trade safety gate** — risk-off, price-impact, deny-by-default, and **depeg risk** (Pegana).
- **Profit-vault safety monitor** — hurdle rate + liquidation buffer, EXIT / DELEVERAGE / ROTATE / HOLD.
- **Default-REJECT backtest rigor** — CPCV / PBO / Deflated Sharpe; honest nulls.
- **Non-custodial custody** — TEE / scoped wallet; withdrawal never gated.
- **Neutral execution adapters** — OKX · Jupiter · Kamino · Drift.

## In progress

- **More grounding data** — beyond peg-risk: venue concentration (CoinGecko), on-chain rug/honeypot checks (QuickNode), with smart-money flow + OHLCV + live research next. See [Data & grounding](/data-grounding).
- **Credits** — a per-account credit balance so you can start with free credits and only pay when you fund. Replaces any access workaround with clean accounting.

## Coming next

- **Fund with Solana Pay** — top up credits from any Solana wallet (QR / link); pay-as-you-go over x402 for agents stays.
- **Trade-agent runtime** — run a strategy locally and let it call the oracle on cadence + triggers (advisor mode first).
- **Listed on pay.sh** — discoverable as a paid verification service for any agent.
- **Multi-chain** — the verification layer is venue/chain-neutral by design; more chains as the adapters land.

<Note>This page is intentionally honest about what's live vs planned. If it's under "coming next," it isn't shipped yet — don't trade on a promise.</Note>
